# LGT MVP Backlog

This backlog keeps the app centered on two authoritative content sources:

- KJV Bible
- Sermons of Rev. William Marrion Branham

## Phase 1: Foundation

1. App shell and navigation
   - Global layout
   - Desktop nav
   - Mobile bottom nav
   - Auth gating
   - Offline banner
   - Sermon audio dock

2. Content infrastructure
   - Prisma schema
   - Seed strategy for Bible books and sermon metadata
   - Shared content types
   - Search indexes

3. Home feed
   - Scripture of the day
   - Branham quote of the day
   - Featured sermon
   - Announcements

## Phase 2: Doctrine

4. Bible reader
   - Book list
   - Chapter reader
   - Verse search
   - Bookmarks
   - Highlights
   - Notes
   - Reading plan

5. Sermon library
   - Search
   - Topic filters
   - Audio player
   - Transcript viewer
   - Scripture references
   - Saved sermons
   - Sermon notes

6. Daily devotion
   - KJV reading path
   - Branham sermon excerpt
   - Reflection prompt
   - Daily streak or progress

## Phase 3: Community

7. Prayer wall
   - Anonymous requests
   - Public intercession feed
   - Answered prayers
   - Moderation queue

8. Testimonies
   - Submission form
   - Approval workflow
   - Featured testimony
   - Comments or reactions

9. Fellowship groups
   - Regional groups
   - Study groups
   - Prayer groups
   - Group posts

10. Events
    - Meeting pages
    - RSVP flow
    - Calendar view
    - Livestream links
    - Reminders

## Phase 4: Utility

11. Locator
    - Assemblies
    - House fellowships
    - Ministers
    - Map and list views

12. Downloads and offline mode
    - Cached sermons
    - Cached Bible content
    - Offline library
    - Sync metadata

13. Profile
    - Saved content
    - Notes
    - Reading progress
    - Prayer interests
    - History

## Phase 5: Stewardship

14. Support
    - Giving
    - Contact
    - Needs board

15. Admin
    - Content moderation
    - Sermon publishing
    - Bible data management
    - User and role management
